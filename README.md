> **Future Interns – Prompt Engineering Internship 2026 | Task 3**

---

## Project Overview

This repository is a complete, production-ready **AI-powered SEO Content Pack** built for a spoken English coaching institute in Chennai. It demonstrates how structured prompt engineering workflows can replace expensive SEO agency retainers — generating keyword strategies, long-form pillar blogs, supporting content clusters, and local SEO adaptations entirely through AI.

The project follows the exact content architecture used by professional SEO agencies:

- One strong **Pillar Blog** targeting the primary keyword
- Four **Supporting Blogs** forming a topical content cluster
- A **Keyword Strategy** document mapping intent and opportunity
- A **Content Cluster Map** with internal linking and publishing order
- **Reusable prompt templates** that can be adapted for any client or city

| File | Purpose |
|---|---|
| `01_keyword_strategy_prompt.md` | Generates keyword lists, search intent, PAA ideas, meta titles, and internal link suggestions |
| `02_pillar_blog_prompt.md` | Produces a 2000+ word pillar article with full SEO structure |
